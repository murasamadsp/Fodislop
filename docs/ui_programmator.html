Динамическая сборка UI из серверных пакетов

При получении пакета OpenWindowPacket интерфейс строится динамически из дерева компонентов в PacketUIBuilderFactory:

  • CanvasPacketBuilder / PanelPacketBuilder — базовые корневые контейнеры.
  • DockPanelPacketBuilder — верстка с докингом компонентов (`DockPanel.Dock` -> `Left`, `Top`, `Right`, `Bottom` через Dock.cs).
  • TextPacketBuilder, ButtonPacketBuilder, SliderPacketBuilder, ScrollViewerPacketBuilder — визуальные элементы UI Toolkit.

Привязка данных WindowBinding (SmartFormat)

Класс WindowBinding реализует авто-связывание данных окон:

  • Сканирует дерево VisualElement на наличие полей ввода (источников) и текстовых полей с шаблонами `SmartFormat`.
  • При изменении значений во входе автоматически пересчитывает форматированные выражения и обновляет связанный интерфейс.

Система Программатора (Programmator)

Программатор предоставляет визуальный граф команд для алгоритмов робота:

Класс Файл Назначение
ProgrammatorGrid ProgrammatorGrid.cs Визуальная сетка ячеек для размещения алгоритмических команд.
ProgrammatorData ProgrammatorData.cs Структура данных алгоритма, сериализуемая для отправки на сервер.
RadialMenu RadialMenu.cs Круговое меню быстрого выбора инструкций робота.
