SingleMeshTerrainRenderer

Процедурный рендерер SingleMeshTerrainRenderer динамически строит один меш на весь видимый экран тайлов, сводя количество вызовов отрисовки (Draw Calls) к 1.

  • Sorting Order: -1000. Рендерится первее всех игровых сущностей.
  • Материал: Единый атласированный 2D URP материал. Все текстуры тайлов упаковываются в TextureAtlas.
  • Буферизация: Пересчёт меша производится при перемещении камеры за пределы порога или при чанковых изменениях MapStorage.

Структура 7 UV-каналов меша

Для каждого квада тайла вершины содержат 7 специализированных UV-каналов:

Канал Данные Назначение
uv0 Vector2 (U, V) Базовые координаты квада в текстурном атласе (TextureAtlas)
uv1 Vector2 (MaskBitmask, Variant) Битмаска авто-тайлинга TileMaskConverter для примыкания соседей
uv2 Vector2 (AnimFrame, AnimSpeed) Индекс текущего кадра анимации и скорость для анимированных тайлов
uv3 Vector2 (ShadowMask, AmbientOcclusion) Маска затенения и эмбиент-окклюзия углов тайла
uv4 Vector2 (ReliefDepth, LightIntensity) Глубина рельефа и уровень освещённости блока
uv5 Vector2 (OverlayU, OverlayV) Текстурные координаты накладываемого слоя (трещины, разрушения)
uv6 Vector2 (CustomFlags, BiomeId) Флаги биологического биома и пользовательских эффектов

SurfaceRenderer (Transit & Perspective)

Для отрисовки поверхностей и перспективы используется SurfaceRenderer:

  • Transit — накладывает плавные градиентные переходы между слоями блоков на разных глубинах.
  • Perspective — достраивает перспективные боковые грани для создания эффекта объёма 2.5D при перемещении камеры.

Проекция Y-координат на карты (Minimap / WorldMap)

При рендеринге текстур в MinimapController и WorldMapRenderer применяется точное сопоставление координат:

  • Серверные координаты: serverY = 0 (поверхность мира, верх). serverY растёт вниз.
  • Текстурные пиксели Texture2D: py = 0 (низ экрана), py = texH - 1 (верх экрана).
  • Формула проекции: float pixelY = ((cy - cellY) / cp) + (texH * 0.5f); — обеспечивает правильную ориентацию севера/юга на экранах без вертикального переворота.
